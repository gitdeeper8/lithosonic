# LITHO-SONIC Core Physics Engine

This module contains the implementation of the five governing parameters:

1. **Biot Coupling Coefficient (B_c)** - `biot.py`
2. **Acoustic Impedance Contrast (Z_c)** - `impedance.py`
3. **Hydraulic Fracture Resonance (f_n)** - `fracture_resonance.py`
4. **Acoustic Attenuation (α_att)** - `attenuation.py`
5. **Acoustic Emission Rate (Ṡ_ae)** - `emission.py`
6. **Lithospheric Stress Index (LSI)** - `lsi.py`

The main interface is the `LithoSonic` class in `litho_physics.py`.
