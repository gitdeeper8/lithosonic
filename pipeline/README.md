# LITHO-SONIC Processing Pipeline

Three-stage signal processing pipeline:

## Stage 1: Spectral Decomposition
- FFT analysis on overlapping 60-second windows
- Power spectral density estimation
- Peak detection

## Stage 2: Harmonic Analysis
- Harmonic series identification
- Fracture end-condition determination
- Mode matching

## Stage 3: Parameter Inversion
- Conjugate gradient optimization
- Monte Carlo uncertainty propagation
- 5-parameter estimation
