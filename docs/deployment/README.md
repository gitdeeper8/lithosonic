# LITHO-SONIC Field Deployment Guide

## Site Selection Criteria

- Low anthropogenic noise (< 1 Hz band)
- Bedrock coupling for geophones
- Power availability (grid or solar)
- Cellular/network connectivity
- Accessibility for maintenance

## Installation Procedure

1. Site survey and GPS positioning
2. Geophone array installation (cross pattern, 50 m spacing)
3. DAS cable burial (0.3 m depth)
4. Borehole drilling for Paros transducer (30 m)
5. Central station setup with STS-5A and MB2005
6. Power system installation
7. Network configuration
8. System testing and calibration

## Maintenance Schedule

- Weekly: Remote data check
- Monthly: Sensor status verification
- Quarterly: On-site inspection
- Annually: Sensor calibration
- As needed: Battery replacement, data download
